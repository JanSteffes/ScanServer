# ScanServer
Java Server to scan/merge/list files using various libs (to minimize fileSize) that have to be preinstalled on a unix system.

Used libs:
* pdftk
* scanimage
* tiff2pdf
* pdftops
* ps2pdf

System used on:

RaspberryPi with os:
